# DESIGN SYSTEM PHILOSOPHY: "Classic Masters"

> [!IMPORTANT]
> **Product Strategy**: "Philosophical Design"
> **Core Theme**: Art meets Data.
> **Constraint**: **Zero-Cost Asset Policy** (No paid AI generation).

## 1. Visual Theme: "The Masters"
We adopt a design language inspired by classical art masters to differentiate from generic "Crypto Dark Mode" sites.

| Element | Inspiration | Application |
| :--- | :--- | :--- |
| **Color** | **Claude Monet** | Impressionist Palettes. Soft gradients, light-responsive hues, calming but vibrant. |
| **Texture** | **Van Gogh** | Visible strokes (subtle patterns), organic curves, emotional resonance in charts. |
| **Structure** | **Da Vinci** | Grid systems, Golden Ratio, anatomical precision in data layout. |

## 2. Brand Identity
*   **Name**: **ChartMaster**
*   **Slogan**: "Art of Data, Science of Profit"
*   **Tone**: Professional, Insightful, Calm (Not Hype-driven).

## 3. Asset Policy (Strict)
*   **External AI APIs**: **FORBIDDEN** (Midjourney/DALL-E via API).
    *   *Reason*: Cost control.
*   **Allowed Assets**:
    *   Hand-picked Public Domain Art (Museum collections).
    *   User-generated assets (manually uploaded).
    *   CSS-generated visuals (Gradients, Shadows, SVG).
    *   Lucide React Icons (Vector).

## 4. Component Rules
1.  **Typography**: Use **Inter** or **Outfit** (Google Fonts). Clean, modern sans-serif to balance the "Art" theme.
2.  **Surfaces**: Glassmorphism (Frosted Glass) to mimic "painting layers".
3.  **Motion**: Smooth, flow-like transitions (Framer Motion). No abrupt jumps.
